
# How to create a public & private SSH Key Pair

1. step 1 in a terminal cd into your .shh/ folder
2. type into terminal = ssh-keygen -t rsa -b 4096 -C "email-address"
3. it will then ask for a name "tech515-oliver-gcp-key"
4. it will then ask for a password twice - just leave blank and click enter and then it should display your random-art image for the key and it will be created
